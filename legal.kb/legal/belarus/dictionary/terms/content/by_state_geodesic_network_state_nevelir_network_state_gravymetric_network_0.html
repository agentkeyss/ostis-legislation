Государственная геодезическая сеть, государственная нивелирная сеть, государственная гравиметрическая сеть – геодезическая, нивелирная, гравиметрическая сети, обеспечивающие распространение установленных на территории Республики Беларусь соответственно государственной системы отсчета координат, государственной системы отсчета высот, государственной системы отсчета гравиметрических измерений и являющиеся исходными для построения других геодезических, нивелирных, гравиметрических сетей.
